# 볼넷
- 1루 주자 x: 1루에 주자 추가
- 1루에 주자 o, 2루 주자 x: 2루에 주자 추가
- 1, 2루에 주자 o, 3루 주자 x: 3루에 주자 추가
- 만루: 주자 상태 유지
---
# 삼진
- 무사 또는 1사: 아웃카운트 1개 추가
- 2사: 이닝 종료
---
# 뜬공: 3루 주자 존재 및 무사 또는 1사
- 리터치 시도 x: 아웃카운트 추가, 주자 상태 유지
- 리터치 시도 성공: 아웃카운트 추가, 3루 주자 삭제
- 리터치 시도 아웃(1사): 이닝 종료
- 리터치 시도 아웃(무사): 아웃카운트 2개 추가 및 3루 주자 삭제
---
# 뜬공: 3루 주자 x
- 무사, 1사: 아웃카운트 추가
- 2사: 이닝 종료
---
# 땅볼: 2사
- 이닝 종료

# 땅볼: 주자 1루, 타자 주자 생존
- 3루 주자: 타구 이전의 2루 주자 상태
- 2루 주자 x
- 1루 주자 o
- 아웃 카운트 추가

# 땅볼: 주자 1루, 1루 주자 생존
- 3루 주자: 타구 이전의 2루 주자 상태
- 2루 주자 o
- 1루 주자 x
- 아웃 카운트 추가
---
# 땅볼: 병살
- 무사: 3루 주자: 타구 이전 2루 주자 상태, 아웃 카운트 2개 추가
- 1사: 이닝 종료
- 주자 1루 x
    - 무사 또는 1사: 아웃 카운트 1개 추가
        - 3루 주자 상태: 타구 이전 2루주자 상태
---
# 단타: 주자 2루 존재
- 1루 주자 상태: 존재
- 2루 주자 상태: 타구 이전의 1루 주자 상태
- 2루 주자 홈 승부 성공: 3루 주자 x
- 2루 주자 홈 승부 실패: 3루 주자 x, 아웃카운트 추가
- 2루 주자 홈 승부 x: 3루 주자 존재
---
# 단타: 주자 1루 시
- 1루 주자 3루 시도: 주자 1, 3루
- 1루 주자 3루 시도 x: 주자 1, 2루
